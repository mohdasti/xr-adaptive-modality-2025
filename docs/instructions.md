# Participant Instructions (Study Handout)

## Goal
Select targets as **fast and accurate** as possible.

## Inputs
- **Hand:** Move cursor and click the target.
- **Gaze-confirm (simulated):** Hover over the target, then press **Space** to confirm.

## Trial Steps
1. Start at the home point.
2. Move to the highlighted target.
3. Hand: click. Gaze-confirm: hover + Space.
4. Proceed to the next trial.

## Environment Requirements
- Chrome desktop, **100% zoom**, **full-screen**.
- Minimum window size 1280×720.
- Close heavy apps; keep a stable position.

## Timing
- 4 short blocks (~4–6 minutes each). You can rest between blocks.

## Questions?
Ask the moderator any time.
